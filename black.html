BlackBox

Según Paul Virilio la creación de una tecnología es también la catastrofe que esa tecnología posibilita. Toda tecnología implica una Escatologia ¿como podria ser una caja negra de las redes sociales? Un espacio donde se pudiese registrar el momento previo a la catastrofe...

BlackBox es una red social local, accesible a través de un punto de acceso Wifi, emitido por un objeto compuesto por un microprocesador en una estructura de acrílico y metal. La red funciona en las inmediaciones del objeto, el cual almacena la interfaz de la red social y Las interacciones de los usuarios. Este artefacto es al mismo tiempo el proveedor de red, el servidor web y la nube. En BlackBox no hay perfiles y solo es posible una participación por dispositivo. La consigna es “para ver un mensaje, deja un mensaje”. Cualquier dispositivo móvil o computador puede conectarse y acceder a la interfaz a través de un navegador. Cada usuario puede dejar un mensaje de 100 caracteres. BlackBox devuelve un mensaje previo seleccionado aleatoriamente. Cuando se ingresa un mensaje el led cambia su pulsación, dando a conocer la interacción. Si se intenta ingresar por segunda vez desde el mismo dispositivo, la interfaz lleva a una pagina de denegación y expulsa al dispositivo de la red.

BlackBox is a local social network, accessible through a Wifi access point, emitted by an object composed of a microprocessor in an acrylic and metal structure. The network works in the proximity of the object, which stores the social network interface and user interactions. This device is at the same time the network provider, the web server and the cloud. In BlackBox there are no profiles and only one participation per device is possible. The slogan is "to see a message, leave a message". Any mobile device or computer can connect and access the interface through a browser. Each user can leave a message of 100 characters. BlackBox returns a previously selected random message. When a message submmits, the led changes its pulsation, making the interaction known. If you try to enter a second time from the same device, the interface leads to a denial page and expels the device from the network.
